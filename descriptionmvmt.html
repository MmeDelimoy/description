Point rouge: mouvement curviligne, vitesse variable (indépendant du repère).
Le repère Oxy, lui, peut être déplacé.

Position actuelle dans le repère (x, y): (0, 0)

Contrôles

Origine du repère: cliquer près de O puis déplacer la souris (repère visible).

Vecteurs


Position, déplacement, autre: 1er clic = origine, 2e clic = extrémité.
Vitesse instantanée: clic sur la trajectoire (le vecteur est calculé).
Vitesse moyenne: 1er clic = point A, 2e clic = point B sur la trajectoire.

Liste des vecteurs

Mesures

Vitesse instantanée: aucune sélection.

Chronomètre (segment pour vitesse moyenne): aucun segment.

Vitesse moyenne: aucune sélection.

Unités: position / déplacement: unités de position (une case = 1 unité).
vitesse: unités de position par seconde.
L’échelle des vecteurs vitesse est choisie pour être visible à l’écran.
